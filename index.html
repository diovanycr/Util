Mensagens

Cadastre respostas prontas para usar no atendimento. Clique no texto para copiar instantaneamente. Arraste para reordenar.

Problemas / Soluções

Cadastre problemas recorrentes e suas soluções. Cada problema pode ter múltiplas soluções. Clique na solução para copiar. Arraste para reordenar.

Imagens nas soluções

Cole imagens diretamente no editor de solução com Ctrl+V. Útil para prints de tela.

Busca global

Use Ctrl+K para buscar em mensagens e problemas ao mesmo tempo. Clique no resultado para copiar ou navegar.

Sistemas

Ferramentas utilitárias integradas. Inclui o Port Opener para gerar scripts de abertura de portas no Firewall do Windows.
